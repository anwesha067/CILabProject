## Installation Guide
1. Install Java (JDK 17)
2. Install Jenkins using Homebrew
3. Start Jenkins using `brew services start jenkins-lts`
4. Access Jenkins at http://localhost:8080

## User Manual
- Create Freestyle and Pipeline jobs
- Configure GitHub repository
- Run builds manually or using triggers
- View build logs and artifacts

## Troubleshooting Guide
| Issue | Solution |
|------|----------|
| Build failed | Check console output |
| Git error | Verify repository URL |
| Email not sent | Check SMTP settings |
| Permission denied | Use admin account |
